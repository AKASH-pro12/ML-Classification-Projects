# ML-Classification-Projects
Machine Learning classification projects including Spam Email Detection and Student Performance Prediction using Python and Scikit-learn.

This repository contains machine learning classification projects developed using **Python and Scikit-learn**.
The objective of these projects is to demonstrate the end-to-end machine learning workflow including **data preprocessing, feature engineering, model training, and evaluation**.

---

## 📌 Projects Included

### 1️⃣ Spam Email Classification

This project focuses on detecting whether an email is **Spam or Not Spam** using Natural Language Processing techniques and machine learning.

### Workflow

* Data loading and preprocessing
* Text feature extraction using **TF-IDF Vectorization**
* Model training using **Random Forest Classifier**
* Model evaluation using **Accuracy Score and Confusion Matrix**

### Input

Email message text

### Output

* **Spam (1)**
* **Not Spam (0)**

---

### 2️⃣ Student Performance Prediction

This project predicts whether a student will **Pass or Fail** based on academic performance data.

### Workflow

* Data preprocessing and feature selection
* Train-test data splitting
* Model training using **Random Forest Classifier**
* Model evaluation using **Accuracy Score and Confusion Matrix**

### Input Features

* Study time
* Number of past failures
* Absences
* Previous grades (G1, G2)

### Output

* **Pass (1)**
* **Fail (0)**
